HTML

  • HyperText Markup Language
  • Linguagem de marcação responsável por definir a estrutura de um documento
  • Conteúdo é anotado por meio de elementos suportados pela linguagem (tags)
  • Navegador recebe documento HTML repleto de tags, as interpreta e as exibe para o usuário

HTML

Elementos

  • São utilizados para descrever a estrutura da página
  • Cada elemento tem um propósito único e específico
  • São descritos por meio de <xml>tags</xml>

<p> Olá, HTML! </p>

Atributos

  • Alguns elementos possuem informações extras que não podem ser exibidas na página
  • Essas informações podem ser definidas por meio dos atributos de um elemento
  • Um elemento pode ter vários atributos definidos

<a href="http://www.google.com.br"> Clique aqui </a>

Elementos Aninhados

  • Elementos podem (e devem!) ser declarados dentro de outros elementos
  • O aninhamento cria uma relação entre os elementos
  • Para isso, deve-se lembrar de fechar os elementos internos antes dos externos, sempre!

<p>HTML é <b>muito</b> legal!</p>

Elementos Vazios

  • Alguns elementos não precisam de conteúdo para serem renderizados
  • Seu "conteúdo" já é pré-definido ou carregado de fontes externas
  • Nesse caso, o elemento é fechado na sua abertura

<img src="images/coltec-ufmg.png" alt="Logo COLTEC" />

Estrutura Básica


        <!DOCTYPE html>
        <html>

            <head>
                <title>Minha primeira página HTML</title>
            </head>

            <body>
                <!-- meu comentário -->
                <p> Essa é a minha primeira página web. </p>
            </body>

        </html>

Estrutura Básica

<!DOCTYPE> Versão do documento HTML
<html> delimita a página HTML
<head> Cabeçalho da página. Contém informações de configuração da página
<body> Corpo da página. Contém toda informação que será exibida para o funcionário

Elementos Textuais

Content is king.
Bill Gates, 1996

Elementos Textuais

  • Grande parte do conteúdo das páginas web é composto por texto.
  • As páginas web foram concebidas para serem lidas!
  • Objetivo: Exibem e organizam o texto da página.

Principais Elementos

  1. Parágrafos
  2. Headings
  3. Listas
  4. Citações

Parágrafos

  • Elemento-base de exibição de texto.
  • Exibe o texto no formato de um parágrafo.
  • Representado pelo elemento <p>.

<p>Olá! Eu sou um parágrafo.</p>

Headings

  • Fornecem os títulos da página.
  • Disponibilizados em seis níveis: <h1> (mais importante) a <h6> (menos importante).

<h2>Subtítulo da página</h2>

Listas

Podem ser definidas três tipos de listagem

  • <ul> lista elementos não-ordenados
  • <ol> lista elementos ordenados
  • <dl> representam listas de definição
            
                <ul>
                    <li>item 1</li>
                    <li>item 2</li>
                    <li>item 3</li>
                </ul>
            
        
            
                <ol>
                    <li>1o item</li>
                    <li>2o item</li>
                    <li>3o item</li>
                </ol>
            
        
            
                <dl>
                    <dd>definição 1</dd>
                    <dd>definição 2</dd>
                    <dd>definição 3</dd>
                </dl>

Formatação de Texto

  • Elementos que modificam a exibição de trechos de texto.
  • Ajudam a definir o significado (semântica) de determinadas palavras na página.
    1. <a> para definir links.
    2. <strong> ou <b> para destaque em negrito.
    3. <i> ou <em> para destaque em itálico.
    4. Muitos outros...
        
            <p>Os elementos textuais são <strong>muito</strong> importantes.</p>
